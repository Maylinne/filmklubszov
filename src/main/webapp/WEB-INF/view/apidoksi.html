Welcome to filmklubszov.hu's REST API documentation!

GET /api/locations?withspots=

Queries a list of locations. The withspots parameter can have two values: "true", or "false." It sets, that the list will contain the list of spots as well, or not.

Parameters:
  • withspots
    • true: Requests the list of locations with the spots.
    • false: Requests the list of locations without the spots.
Returns:

A json object.

GET /api/locations?id={id}&withspots=

Requests a specific location by its id. The withspots parameter can have two values: "true", or "false." It sets, that the list will contain the list of spots as well, or not.

Parameters:
  • id
    • number: Sets the id of requested location.
  • withspots
    • true: Requests the list of locations with the spots.
    • false: Requests the list of locations without the spots.
Returns:

A json object.

POST /api/location

Adds a single location. Can be sent only with spot data.

Accepts:

A json object.

POST /api/spot

Adds a single spot. The data contains the id of the location.

Accepts:

A json object.

POST /api/movie

Adds a single movie. Can be sent only with cut data.

Accepts:

A json object.

POST /api/cut

Adds a single cut. The data contains the id of the movie.

Accepts:

A json object.
